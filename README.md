# discord-bot
# Discord Bot Commands 🔹

Below is a list of all slash commands available on this Discord bot, with their parameters and functionality. Emojis are used to improve readability.

| Command       | Parameters                  | Description |
|---------------|----------------------------|------------|
| 👋 /hello        | –                          | Greets the user who invoked the command. |
| 📜 /rules        | –                          | Displays the server rules. |
| 🏓 /ping         | –                          | Checks if the bot is online and shows latency in milliseconds. |
| ℹ️ /info         | –                          | Shows basic info about the bot (name, ID, number of servers). |
| 🏢 /server       | –                          | Shows information about the current server (name, ID, number of members). |
| 🖼️ /avatar       | <user> (optional)           | Displays the avatar of the specified user or yourself if no user is provided. |
| ⏱️ /uptime       | –                          | Shows how long the bot has been online (days, hours, minutes, seconds). |
| 👤 /userinfo     | <user> (optional)           | Shows detailed info about a user: username, ID, account creation date, join date, roles, avatar. |
| 🎭 /roles        | <user> (optional)           | Shows all roles of the specified user or yourself. |
| 🏛️ /servericon   | –                          | Displays the server icon. |
| 🤖 /botinfo      | –                          | Shows technical info about the bot: version, uptime, number of servers. |
| 🆘 /help         | –                          | Displays a list of all available slash commands with short descriptions. |
| 🔔 /remind       | <time> <message>            | Sets a reminder; the bot sends the message after the specified time (e.g., `10s`, `5m`, `2h`, `1d`). |
| 📊 /poll         | <question> <option1> <option2> | Creates a simple poll with two options; users can vote using reactions 1️⃣ and 2️⃣. |
| 📈 /serverstats  | –                          | Shows basic server statistics: total members, online members, number of text and voice channels. |
| 🛡️ /moderate     | <action> <user>             | Performs basic moderation: kick or ban a user if the bot has permission and hierarchy allows. |
